# From-scratch Naive Bayes Spam Detection Web App and Chrome Extension

This project implements a spam detection web application and a Chrome extension using the Naive Bayes algorithm. The web app serves as an interface for users to check if certain messages are spam, leveraging machine learning techniques to classify messages based on training data. The Chrome extension provides similar functionality directly within the browser, enhancing user experience and convenience.

## Features
- **Web Application**: A user-friendly interface to input text and receive spam classification.
- **Chrome Extension**: Easily check messages while browsing.
- **Machine Learning**: Implements the Naive Bayes classification algorithm for text categorization.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/brookemstevens/brooke-stevens-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd brooke-stevens-portfolio/From-scratch Naive Bayes spam detection web app and Chrome extension
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To run the web application:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) in your browser to use the app.
